## 🚀 快速开始

### 步骤 1：注册账号

访问 [tvbot.cc](https://tvbot.cc) 注册您的专属账号

### 步骤 2：连接交易所

1. 在交易所申请 API Key（**务必只开启"交易"和"只读"权限，禁用提现权限**）
2. 在 TVBot 后台绑定您的 API Key
3. 选择交易对和交易模式

### 步骤 3：配置 TradingView 策略

1. 在 TradingView 中选择您的交易策略或指标
2. 创建警报（Alert），勾选"Webhook URL"
3. 复制 TVBot 提供的 Webhook URL 和消息模板
4. 粘贴到 TradingView 警报配置中

### 步骤 4：启动交易

- 保存配置，启动机器人
- 实时监控交易状态
- 随时调整参数和策略

##❓ 常见问题
💻 需要一直开着电脑吗？
答： 不需要！TVBot 采用云端托管，警报对接完成后，您可以关闭电脑和网页，机器人 24/7 自动运行。
📉 信号触发了但没下单怎么办？
答： 检查以下三点：
持仓模式：确保交易所已改为"买卖模式"（非对冲模式）
保证金：确保账户余额充足
交易对：确认 TradingView 和机器人选择的交易对一致
###🛑 如何停止机器人？
答： 直接在 TradingView 中删除对应的警报即可停止机器人运行。
🔄 如何修改策略参数？
答： 删除现有警报 → 调整策略参数 → 重新添加警报。
📊 可以同时运行多个策略吗？
答： 可以！支持无限个策略同时运行，但注意：一个币种同一时间只能运行一个策略，避免仓位冲突。
